# Layal Theme

**ثيم ليال** هو ثيم أنيق مصمم خصيصاً لمتاجر المجوهرات، الإكسسوارات، والهدايا. يتميز بتصميم بسيط، أنثوي، وسهل التصفح.

## المميزات:
- متوافق مع الجوال
- بنر رئيسي قابل للتخصيص
- صفحة منتج أنيقة مع زر سلة واضح
- دعم عرض الصور والتفاصيل الجذابة
- فوتر واضح وروابط منظمة

## التخصيص:
يمكن لصاحب المتجر تعديل:
- العنوان الرئيسي
- الوصف
- صورة البنر
- نص الفوتر
- خيارات عرض المنتج
